# 企业微信群聊天-每日简报推送脚本

部署vercel serverless

## Configuration
`process.env.NEWS_WECHAT_WEBHOOK_URL` 企业微信群机器人地址
```sh
export NEWS_WECHAT_WEBHOOK_URL="https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=examplekey'"
```

## Installation
```sh
pnpm install
```

## Development
```sh
pnpm start
```
